1. Compile listed sources to the Build folder:
> javac -d Build Hello.java Server.java Client.java

2. Compile all sources listed in the sources.txt file to the Build folder:
> javac -d Build @sources.txt

3. Create a list of all .java files and send to the source.txt file:
> dir /s /B *.java > sources.txt

4. Start the registry (must run from the Build folder)
> start rmiregistry

5. Start the server
> start java -classpath Build -Djava.security.policy=server.policy -Djava.rmi.server.codebase=file:Build/ ct414.ExamEngine

6. Start the client
> start java -classpath Build ct414.ExamClient
